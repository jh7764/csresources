# CS Resources

A student-built index of computer science tools, courses and sites. Whether you're just starting out or prepping for technical interviews, this site has something for you.

# What's Inside

- **Learning Resources** — all-in-one platforms like GeeksForGeeks and W3Schools
- **Courses & Certifications** — free courses from Full Stack Open, freeCodeCamp, Cisco, and more
- **Roadmaps** — guided learning paths for different CS tracks
- **Coding Practice** — LeetCode, Codewars, Exercism, and others for sharpening your skills
- **Course Specific Tools** — tools tied to specific UMass Boston CS courses (CS310, CS420)
- **AI Tools** — AI-powered tools built for developers
- **Libraries & Frameworks** — React, Django, TensorFlow, and more
- **API Dev & Testing Tools** — Postman, Swagger
- **Other Tools** — debuggers, flashers, and other utilities

## Tech Stack

- **React** — component-based UI
- **CSS** — custom styling
- **JavaScript**
